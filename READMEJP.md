# NimPL

注意 : このプロジェクトは開発段階( Alpha )です。うまく利用できない可能性があります。

NimPLは、軽量でシンプルな純Nim製のGUIグラフツールです。<br>
(PythonのMatplotlibを意識しています。)

### 機能

< 開発中 ><br>
折れ線グラフ (マーカーあり|なし)<br>

< まって ><br>
ヒストグラム<br>
円グラフ<br>
散布図<br>
バブル図<br>

< 議論中 ><br>
面グラフ<br>
等高線<br>

### 対応予定プラットフォーム
Windows ( Win32API ) < 開発中 ><br>
Linux ( GTK+ 3 ) < まって ><br>
MacOS ( GTK+ 3 ) < まって ><br>
